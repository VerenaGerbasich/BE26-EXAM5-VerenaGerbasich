## Features

- **Full CRUD (Create, Read, Update, Delete)** management for Animal records - accessible only to administrators.
- **Role-Based Access Control** with separate permissions for regular users and administrators.
- **User Authentication & Registration** with secure password hashing and session management.
- **Advanced Filtering** - dedicated Senior Animals page for pets older than 8 years.
- **Pet Adoption System** with adoption requests tracked in a dedicated database table with foreign key relationships.
- **Secure Image Upload** supporting both file uploads and URL-based images for animals and user profiles.
- **Admin Dashboard** for complete animal and adoption request management.
- **User Profile Page** displaying personal information and adoption history.
- **Responsive Design** with modern purple-themed UI and gradient effects.

---

## Tech Stack

- **PHP** (Server-side logic, authentication, and database operations)
- **MySQL** (Database persistence via XAMPP/localhost with MySQLi procedural style)
- **HTML5 and CSS3**
- **Modern CSS** with custom gradients, animations, and responsive layouts
- **Session Management** for user authentication and role-based access control
